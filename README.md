# Homebrew Tap for Clank CLI

Install the [Clank](https://clank.host) CLI via Homebrew.

## Install

```bash
brew tap clankhost/clank
brew install clank
```

## Update

```bash
brew upgrade clank
```

## About

Clank is an open-source deployment platform. The CLI lets you deploy, manage services, and stream logs from your terminal.

- [Documentation](https://docs.clank.host)
- [GitHub](https://github.com/clankhost/clank-cli)
